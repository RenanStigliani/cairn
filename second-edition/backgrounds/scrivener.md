# Escrivaneiro

> Você copia textos antigos e ilumina manuscritos, registrando as vozes dos espertos, dos grandes e dos poderosos. Você provará que a pena é realmente mais poderosa que a espada.

## Nomes

Lazlo, Stilo, Akshara, Pisa, Ji-Yun, Kalamos, Hugo, Shui, Kalam, Julius

## Equipamento Inicial

- 3d6 Peças de Ouro
- Rações (3 usos)
- Tocha (3 usos) 
- Pena e Tinta
- Livro em Branco
- Sovela (d6)
- Distintivo (_miudeza_)

## Que trabalho você guardou para si mesmo? Role 1d6.

|       |                                                                                                                                                                                                                                        |
| ----- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **A Língua Selvagem**. Um pacote de pergaminhos encadernados em couro. Uma obra seminal, catalogando as línguas ocultas das bestas e como entendê-las.                                                                                 |
| **2** | **A Sinfonia Silenciosa**. Encadernado em invólucro fluorescente. Muito raro, narra os sinais sutis usados por aqueles que empregam magia de invisibilidade.                                                                           |
| **3** | **Um Tratado sobre o Abismo**. Um livro preto indescritível. Um texto aprofundado e amplamente teórico descrevendo as **Raízes**, bem como informações sobre a localização de um **Portão** próximo.                                   |
| **4** | **A Valsa das Estrelas**. Um fecho em forma de cometa encadernado em uma capa de couro fino. Cartas astronômicas detalhadas, movimentos celestes e festivais estelares. Altamente valorizado (100po) por sua utilidade para viajantes. |
| **5** | **A Catedral e o Dossel**. Encadernação de folhas grandes sobre velino. Nominalmente um livro de histórias infantis, as margens detalham informações sobre viajar, comer e dormir nas florestas nubladas.                              |
| **6** | **Jardim de Vidro**. Encadernado na capa de outro livro. Uma obra herética, descreve os materiais, procedimentos e locais ideais necessários para abrir um **Portão**.                                                                 |

## Como você transcreve informações confidenciais? Role 1d6.

|       |                        |                                                                                                                          |
| ----- | ---------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **1** | **Tinta da Mentira**   | Brilha quando usada para escrever declarações verdadeiras, mas desaparece se usada para escrever falsas.                 |
| **2** | **Pedra de Cifra**     | Um par de pedras negras afiadas. Cada uma descriptografa qualquer mensagem escrita pela outra.                           |
| **3** | **Pena Eterna**        | Uma pena que escreve em qualquer superfície. Você ainda precisa de tinta. _Miudeza_.                                     |
| **4** | **Frasco de Sussurro** | Sussurre uma mensagem no frasco, e ele a reproduzirá para quem o abrir em seguida.                                       |
| **5** | **Lente Sanguínea**    | Extrai sangue de um alvo sem o conhecimento dele. Uma gota roubada colocada no olho revela memórias do dia anterior.     |
| **6** | **Folha de Eco**       | Um pergaminho em branco. Quem o desenrola vê suas ações do dia lentamente reveladas em um garrancho apertado. _Miudeza_. |
